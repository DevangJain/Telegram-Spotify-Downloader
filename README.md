# |-------------------------------------------------->💋Use: 
* PERSONAL ---- Copy any Valid Spotify link and type /spotify "URL" inside the bot and wait.

**THE BIGGER THE SIZE,THE MORE TIME YOU HAVE TO WAIT**
**File size more then 2gb won't be downloaded due to telegram policy**
